# 🍕 Pizza Order WinForms

A simple and interactive **C# Windows Forms (WinForms)** application that allows users to customize their pizza order and view the total price in real time.

This project demonstrates the fundamentals of **event-driven programming**, **GUI development**, and **dynamic price calculation** using **C#** and **WinForms**.

---

## ✨ Features

* 📏 Choose your pizza size

  * 🍕 Small
  * 🍕 Medium
  * 🍕 Large

* 🧀 Select your favorite toppings

  * Extra Cheese
  * Onion
  * Mushrooms
  * Olives
  * Tomatoes
  * Green Peppers

* 🍞 Choose a crust type

  * Thin Crust
  * Thick Crust

* 🥡 Select your order type

  * Eat In
  * Take Out

* 💰 Automatic price calculation

* 📋 Live order summary

* ✅ Order confirmation dialog

* 🔄 Reset the order with one click

* 🖥️ Simple and user-friendly interface

## 🚀 How to Use

1. 🍕 Select a pizza size.
2. 🧀 Add your favorite toppings.
3. 🍞 Choose the crust type.
4. 🥡 Select **Eat In** or **Take Out**.
5. 💵 Watch the total price update automatically.
6. ✅ Click **Order Pizza** to place your order.
7. 🔄 Press **Reset** to start a new order.
